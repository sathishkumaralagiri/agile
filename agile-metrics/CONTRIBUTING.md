# Contributing to agile-metrics

Thank you for your interest in contributing! This is a community-maintained reference library.

## How to Contribute

### Reporting Issues
- Use GitHub Issues for typos, incorrect formulas, outdated content, or missing topics.
- Label issues with: `bug`, `enhancement`, `new-metric`, `new-technique`, `translation`.

### Submitting Content

1. **Fork** the repository
2. **Create a branch**: `git checkout -b feat/add-pert-template`
3. **Make changes** — follow the style guide below
4. **Open a Pull Request** with a clear description

### Style Guide

- Use plain Markdown (no custom HTML)
- Formulas in fenced code blocks
- Tables for comparisons
- Include: Definition → Formula → Example → When to use → ⚠️ Warnings
- Cite sources where possible

### What We Accept

✅ New metric definitions  
✅ Estimation technique comparisons  
✅ Formula corrections  
✅ Real-world examples  
✅ Tool mentions (non-promotional)  
✅ Translations (add `/docs/i18n/<lang>/`)  

❌ Vendor promotional content  
❌ Opinion pieces without evidence  
❌ Duplicate content  

## Code of Conduct

Be kind. Be constructive. Be evidence-based.
