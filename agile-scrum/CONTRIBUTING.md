# Contributing to agile-scrum

Thank you for your interest in contributing! This is a community-maintained reference library.

## How to Contribute

### Reporting Issues
- Use GitHub Issues for typos, incorrect information, outdated content, or missing topics.
- Label issues with: `bug`, `enhancement`, `new-content`, `translation`.

### Submitting Content

1. **Fork** the repository
2. **Create a branch**: `git checkout -b feat/add-sprint-goal-examples`
3. **Make changes** — follow the style guide below
4. **Open a Pull Request** with a clear description

### Style Guide

- Use plain Markdown (no custom HTML)
- Include: Definition → Responsibilities/Rules → Example → ⚠️ Warnings
- Keep examples practical and team-focused
- Cite the Scrum Guide where relevant
- No vendor-specific content (no Jira how-tos, etc.)

### What We Accept

✅ Clarifications to role descriptions
✅ Additional anti-pattern examples
✅ Real-world facilitation tips
✅ Template additions in `/docs/templates/`
✅ Translations (add `/docs/i18n/<lang>/`)

❌ Vendor promotional content
❌ Methodology opinion pieces without evidence
❌ Duplicate content

## Code of Conduct

Be kind. Be constructive. Be evidence-based.
