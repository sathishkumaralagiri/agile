# Contributing to agile-kanban

Thank you for contributing! This is a community-maintained reference library.

## How to Contribute

1. **Fork** the repository
2. **Create a branch**: `git checkout -b feat/add-cfd-template`
3. **Make changes** — follow the style guide below
4. **Open a Pull Request** with a clear description

## Style Guide

- Use plain Markdown (no custom HTML)
- Include: Definition → Rules/Formula → Example → ⚠️ Warnings
- Keep examples practical and team-focused
- No vendor-specific tool tutorials

## What We Accept

✅ Additional CoS examples by industry
✅ Flow metric templates in `/docs/templates/`
✅ Real-world WIP limit case studies
✅ Translations (`/docs/i18n/<lang>/`)

❌ Vendor promotional content
❌ Opinions presented as facts

## Code of Conduct

Be kind. Be constructive. Be evidence-based.
